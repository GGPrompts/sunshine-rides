# Canvas Design Projects

This directory contains completed canvas design projects created using the canvas-design skill.

## Projects

### 1. Memory Force
**Location:** `memory-force/`
**Philosophy:** Memory as Physical Force - treating memory as an invisible gravitational field
**Files:**
- `memory_force_philosophy.md` - Design philosophy document
- `create_memory_force_poster.py` - Generation script
- `memory_force_poster.pdf` - Final PDF output
- `memory_force_poster.png` - Final PNG output

### 2. Mnemonic Fields
**Location:** `mnemonic-fields/`
**Philosophy:** Visual exploration of memory and mnemonic systems
**Files:**
- `mnemonic-fields-philosophy.md` - Design philosophy document
- `create_poster.py` - Generation script
- Multiple iterations:
  - `mnemonic-fields.png/pdf` - Initial version
  - `mnemonic-fields-v1.png` - First iteration
  - `mnemonic-fields-refined.svg` - Refined SVG version
  - `mnemonic-fields-masterpiece.svg` - Masterpiece version
  - `mnemonic-fields-final.png/pdf` - Final versions

### 3. Gravitational Recall
**Location:** `gravitational-recall/`
**Philosophy:** Memory systems represented through gravitational metaphors
**Files:**
- `gravitational-recall-philosophy.md` - Design philosophy document
- `create_gravitational_recall.py` - Initial generation script
- `create_gravitational_recall_svg.py` - SVG version script
- `create_gravitational_recall_refined.py` - Refined version script
- Multiple outputs:
  - `gravitational-recall.png/svg` - Initial versions
  - `gravitational-recall-refined.png/svg` - Refined versions

## Dependencies

All projects require:
- Python 3.x
- Pillow (PIL) library

Install dependencies with:
```bash
pip install -r ../requirements.txt
```

Or with the venv:
```bash
source ../.venv/bin/activate
pip install -r ../requirements.txt
```

## Usage

Each project can be regenerated by running its Python script:
```bash
python3 create_*.py
```

The scripts will generate the visual outputs in the same directory.
